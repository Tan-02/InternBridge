 InternBridge — Internship Management Platform

A full-stack web app connecting students, companies, faculty and admins for seamless internship management.

Tech Stack
- **Frontend** — HTML5, CSS3, JavaScript
- **Backend** — Node.js, Express.js
- **Database** — MongoDB + Mongoose
- **Auth** — JWT

 Features
- Students can browse & apply to internships
- Companies can post internships & shortlist candidates
- Faculty can track student progress & add notes
- Admin can view placement records & generate reports

Setup
1. Clone the repo
2. Run `npm install`
3. Add `.env` file with `MONGO_URI`, `PORT`, `JWT_SECRET`
4. Run `node Back-End/server.js`
5. Open `Front-end/index.html` with Live Server
